# Dam Safety Compliance Platform — Independent Startup

**Mission:** Make India's dams safe and compliant through affordable IoT monitoring and automated reporting.

**Approach:** Start with one dam, one section, minimal sensors. Prove value. Expand.

---

## The Opportunity

- **5,334 large dams** in India, <15% have adequate monitoring
- **Dam Safety Act 2021** mandates real-time instrumentation — most states can't comply
- **No one** currently offers affordable compliance-as-a-service for dams
- **INR 600-2,500 Cr** addressable market for instrumentation alone

## Starting Point: Gosikhurd Dam, Nagpur

| Parameter | Details |
|-----------|---------|
| Location | Bhandara District, 60 km from Nagpur |
| Type | Earthen dam, 11.35 km long |
| Why here | INR 25,972 Cr invested, only 14% irrigated; existing sensors partially functional; national visibility |
| Initial scope | 2 km critical section, 15-20 sensors |
| Administering body | VIDC (Vidarbha Irrigation Development Corporation), Nagpur |

## Strategy: Start Small, Build On It

```
Phase 0 (6 mo)     Phase 1 (12 mo)      Phase 2 (18 mo)       Phase 3 (18 mo+)
───────────────     ────────────────      ────────────────      ─────────────────
Validate &          15-20 sensors on      Expand Gosikhurd      Second dam +
fundraise           2 km section          + ML models           multi-state
INR 15L             INR 50-80L            INR 1.5-2 Cr         Series A funded
```
