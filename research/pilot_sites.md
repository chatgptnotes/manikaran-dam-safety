# Pilot Dam Research — Nagpur Region

## 1. Primary Candidate: Gosikhurd Dam

| Parameter | Details |
|-----------|---------|
| **Location** | Pauni, Bhandara District (~60 km from Nagpur) |
| **River** | Wainganga (Godavari basin) |
| **Type** | Earthen dam with masonry spillway |
| **Length** | 11.35 km (one of India's longest earthen dams) |
| **Gross Storage** | 1,341 MCM |
| **Command Area (Approved)** | 2,50,800 ha (revised to 1,96,600 ha) |
| **Area Actually Irrigated** | ~35,000 ha (only 14% of potential) |
| **Project Cost (Revised)** | INR 25,972 crore recently approved for completion |
| **Existing Instrumentation** | Encardio Rite sensors (piezometers, settlement gauges) — partially functional |
| **Administering Agency** | VIDC, Executive Director, Nagpur |
| **National Monitoring** | CWC monitors as national project; reviewed by CDSRP-2 |

### Why Gosikhurd is Ideal

- **Maximum impact:** INR 25,972 crore invested, only 14% command area irrigated — platform can directly improve utilization
- **Existing sensor base:** Encardio Rite instrumentation installed, reducing deployment cost
- **Earthen dam = higher risk:** 11.35 km embankment requires continuous seepage and settlement monitoring
- **Political visibility:** Nationally monitored, PM-level attention ensures stakeholder willingness
- **Canal optimization opportunity:** vast gap between potential and actual irrigation creates clear ROI

## 2. Secondary Candidate: Totladoh Dam (Pench Project)

| Parameter | Details |
|-----------|---------|
| **Location** | Pench River, ~80 km north of Nagpur (Maharashtra-MP border) |
| **Type** | Concrete gravity dam |
| **Height / Length** | 74.5 m / 680 m |
| **Gross Storage** | 1,241 MCM (Live: 1,091 MCM) |
| **Spillway** | 14 radial gates |
| **Irrigation Command Area** | 1,04,476 ha (Nagpur & Bhandara via Left & Right Bank Canals) |
| **Drinking Water Supply** | Primary source for Nagpur city (NMC allotment: 540 MLD, actual drawl: ~700 MLD) |
| **Thermal Power** | Cooling water to Koradi & Khaparkheda thermal power stations |
| **Key Issue** | NMC over-draws by ~160 MLD, reducing irrigation by 8,658 ha annually |

### Why Totladoh Complements Gosikhurd

- **Different dam type:** concrete gravity vs. earthen — validates platform across both types
- **Multi-purpose conflict:** drinking-water vs. irrigation vs. thermal-power conflict is quantifiable with real-time data
- **14 spillway gates:** gate operation optimization is a high-value use case for AI-based flood management
- **Proximity:** both dams within 80 km of proposed operations center

## 3. Site Selection Matrix

| Criterion | Gosikhurd | Totladoh |
|-----------|-----------|----------|
| Proximity to Nagpur | 60 km | 80 km |
| Dam type coverage | Earthen | Concrete gravity |
| Existing instrumentation | Partial (Encardio Rite) | Minimal |
| Irrigation impact (ha) | 2,50,800 (potential) | 1,04,476 |
| Multi-stakeholder complexity | Medium (VIDC + CWC) | High (VIDC + NMC + MAHAGENCO) |
| National/political visibility | Very High | High |
| Dam Safety Act compliance urgency | High | High |
| Revenue model validation | Irrigation optimization | Water allocation arbitration |

## 4. The Vidarbha Context

Vidarbha (eastern Maharashtra, 11 districts including Nagpur) is critical because:

- **Farmer suicides:** highest concentration in India, water scarcity is a primary driver
- **Incomplete irrigation projects:** VIDC manages dozens of projects where canals remain unbuilt decades after dam completion
- **Political sensitivity:** Gosikhurd debated in Parliament multiple times; PM approved INR 25,972 crore
- **Institutional capacity gap:** VIDC (HQ: Nagpur) lacks digital monitoring capabilities
